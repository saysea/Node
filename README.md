### 电脑使用

<img src="https://cdn.honeyshaddock.fun/ng1.png" width="500" >

> 1、解压软件包后，直接打开 `v2rayN.exe` 就可以使用，复制好节点信息后，点击左上角服务器，选择从粘贴版导入.

<img src="https://cdn.honeyshaddock.fun/ng4.png" width="500" >

> 2、v2rayN.exe打开后，也可以右击电脑右下角的图标，选择从粘贴版导入.


<img src="https://cdn.honeyshaddock.fun/ng2.png" width="500" >

> 3、成功导入节点信息后，我们可以右击电脑右下角的图标选择服务器选择一个节点.


<img src="https://cdn.honeyshaddock.fun/ng3.png" width="500" >

> 4、选择好节点后，右击电脑右下角的图标选择PAC代理模式，然后就可以愉快的玩耍了.


<img src="https://cdn.honeyshaddock.fun/ng5.png" width="500" >

> 5、我可以通过选中所有的节点，右击用ping来查看节点延迟.
